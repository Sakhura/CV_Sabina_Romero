# Sabina Romero Rodríguez — CV Web

Portafolio profesional / CV web construido con HTML, CSS y JavaScript puro.

## 🚀 Ver online
Una vez publicado en GitHub Pages: `https://TU_USUARIO.github.io/sabina-cv`

## 📁 Archivos
- `index.html` — estructura del sitio
- `style.css` — estilos y diseño visual
- `script.js` — animaciones e interacciones

## 🛠️ Publicar en GitHub Pages (paso a paso)

1. Crea un repositorio en GitHub (ej: `sabina-cv`)
2. Sube estos 3 archivos: `index.html`, `style.css`, `script.js`
3. Ve a **Settings → Pages**
4. En **Source** selecciona: `Deploy from a branch`
5. Branch: `main` → Folder: `/ (root)` → Guardar
6. Espera 1-2 minutos y tu sitio estará en:
   `https://TU_USUARIO.github.io/sabina-cv`

## ✏️ Personalizar
- Actualiza datos de contacto en `index.html`
- Cambia colores editando las variables CSS en `style.css` (sección `:root`)
